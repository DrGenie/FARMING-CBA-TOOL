How to use this decision aid

1. Import data

Go to Data Import and either upload the tab‑delimited dataset file or paste the data. You can also upload or paste the data dictionary (CSV). The tool validates and previews before you commit the dataset.

2. Configure

Open Configuration to set the analysis horizon, discount rates, grain prices, yield persistence patterns, and cost recurrence per treatment. Save scenarios to your browser and load them later.

3. Review results

Return to Results to see the control‑centric table, leaderboard, filters, and a plain‑language narrative. Export cleaned data, treatment summaries, a full sensitivity grid, or an XLSX workbook.

This tool computes replicate‑specific control baselines and applies an explicit amendment input cost scaling rule. Technical details appear in the Appendix.
